Perception–Awareness–Decision (PAD)

Socially-Aware Robot Navigation and Interaction

Edison Jair Bejarano Sepúlveda · Valerio Bo · Alberto Sanfeliu · Anais Garrell
CSIC–UPC (IRI), Barcelona
Paper (PDF) Code Author Page

Abstract

Framework: PAD
Topic: Social Nav + HRI

We present a Perception–Awareness–Decision (PAD) framework for socially-aware robot navigation in shared human environments. PAD integrates geometric navigation with multimodal perception, semantic situational awareness, and context-grounded dialogue, enabling robots to resolve navigation conflicts that require human cooperation.
